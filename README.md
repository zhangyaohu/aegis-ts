#项目简介：
此项目是aegis数据库监控平台、改造为ts+vue的项目后端跟aegis项目相同在下:
https://github.com/zhangyaohu/aegis/service/server目录下运行
yarn run start
注：需要在本地导入数据库文件aegis.sql

前端框架: elementui + vue + typescript + vue-router + echarts
